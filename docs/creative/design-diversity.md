# Assessment Form Design Diversity

- Baseline to avoid: centered hero, big white card, stacked explanatory copy, soft purple/cyan gradient panel, and bottom-only navigation.
- Layout archetype: command-center tool surface.
- Palette and material: brand violet/cyan stay as accents; primary surfaces are white, mist, ink, fine borders, and a subtle grid texture.
- Typography: Poppins for compact headings, Roboto for body and answer text; no hero-scale type inside the assessment tool.
- Asset strategy: no heavy media; use lucide icons, small badges, progress markers, and domain cues.
- Signature interaction: calibration sweep progress plus selected answer feedback.
- Differentiation proof: the redesigned form reads as a task cockpit instead of another landing-page card stack.
- Details intentionally avoided: Awwwards-style page transitions, hover-only reveals, parallax, and decorative form visuals that do not help completion.
