# Assessment Form Creative Brief

- Project name: LitmusAI assessment cockpit
- Audience: learners and reviewers checking AI readiness before training.
- Primary goal: start and complete a 15-question assessment without the form feeling cramped, generic, or tiring.
- Brand personality: sharp, practical, focused, quietly premium.
- Core message: this is a fast calibration, not a long quiz funnel.
- Emotional target: calm momentum and clear feedback.
- Content requirements: level selection, 15 questions, progress, domain label, answer choices, next/submit action, existing save/results behavior.
- Interaction ambition: compact motion that orients the learner and confirms state changes.
- Technical constraints: React, Vite, Tailwind, existing GSAP utility, no new dependencies, preserve API/storage/test contracts.
- Accessibility constraints: keyboard-friendly controls, visible focus, readable type, reduced-motion support, no hover-only information.
- Assumptions: results page remains mostly unchanged; the redesign focuses on intro and active question form.
